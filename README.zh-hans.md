## 3.0 的特性以及如何升级
3.0 提供了两个函数 `client_api` 和 `server_api`，他们可以处理大部分支付宝的接口。

3.0 保持向前的兼容性，唯一需要注意的是 `Alipay.verify` 函数不会将 `sign` 从 `data` 里面 `pop` 出来，你需要自己处理。

## 测试
```bash
python -m unittest discover
```

或者你可以传入 `debug=True`，进行手动测试：
```python
alipay = AliPay(..., debug=True)
```
